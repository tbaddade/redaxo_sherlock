Sherlock
============================================


Stellt einen Finder bereit der
--------------------------------------------

* alle Extension Points listet (auch von aktivierten AddOns)
* alle REX_VARS listet
* alle Artikel eines Moduls anzeigt
* alle Artikel eines Templates anzeigt


Voraussetzungen
-----------------
**REDAXO** 4.5
**PHP:** 5.4
